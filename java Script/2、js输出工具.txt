前言：js操控浏览器的能力
JavaScript的发明目的，就是作为浏览器的内置脚本语言，为网页开发者提供操控浏览器的能力。它是目前唯一一种通用的浏览器脚本语言，所有浏览器都支持。它可以让网页呈现各种特殊效果，为用户提供良好的互动体验。
目前，全世界几乎所有网页都使用JavaScript。如果不用，网站的易用性和使用效率将大打折扣，无法成为操作便利、对用户友好的网站。
对于一个互联网开发者来说，如果你想提供漂亮的网页、令用户满意的上网体验、各种基于浏览器的便捷功能、前后端之间紧密高效的联系，JavaScript是必不可少的工具。

1、js能用来干什么
(1)数据的验证；
(2)将动态的文本内容写入到网页当中；
(3)可以对事件做出响应；
(4)读写html中的内容；
(5)可以检测浏览器；
(6)创建cookies；

2、输出工具；
（1）alert：弹出框；（括号里的东西会全部弹出）【警告】
（2）document.write():在文档中输出；（不会输出标签对， 但是，会继承标签的属性；）
（3）console.log()：在控制台输出；
（4）prompt:弹出框；【确认】
      格式：var aa=prompt("请输入成绩","aaa")
【第一个引号：标题大意；第二个引号：单行文本的默认值；】
（5）confirm：弹出框；【提问true/false】 
     格式： var aa=confirm("你喜欢js吗?")
	alert(aa)
