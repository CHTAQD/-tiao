事实，但这并不意味着以深度报道为主的长新闻就没有前途了：首先，在斯达克曼的研究中，也并不是所有的主流媒体都在减少长新闻。以《纽约时报》为例，尽管2000单词以上的新闻减少了25%，但3000单词以
上的新闻稿件反而增加了32%。这说明，传统媒体上长新闻也有两种不同的走势。其次，在新的移动终端上，长新闻依然是受欢迎的。美国皮尤研究中心2012年报告指出，美国73%的平 板电脑用户阅读长新闻；61%的用户至少阅读过一
篇以上长新闻；超过70%的用户认为，是平板电脑让他们接触到之前不会阅读的长新闻。此外，美国出版者联盟的数据也表明，去年上半年美国电子书销售量增幅达188%，这说明“长阅读”的未来是光明的。因此，他们认为，平板电脑等新移动终端可以“拯救”深度报道。

够说服对方或者取得压倒性的优势。长好？短好？似乎成为了一个问题。 　　其实，跳出双方纠结的数据泥潭来看，长与短，都不过是外
