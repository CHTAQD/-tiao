javascript数据类型隐式转换

一、函数类
    isNaN()
    该函数会对参数进行隐式的Number()转换，如果转换不成功则返回true;则反；
   
二、运算符类
   1.算数运算符
   -  *   /  % 
（1）如果操作数不是数值，将会隐式的调用Number()函数，按照这个函数的转换规则进行转换，
 （2）如果转换不成功，整个表达式返回NaN

   +
（1）  如果操作数都是数值，然后进行相加
（2） 如果操作数都是布尔值，那么进行Number()转换，false为0，true为1，进行相加。
（3）如果不能转换为数值，将表达式的值连接起来；

2.等性运算符   ==     !=
会对操作数隐式的转换后再比较值
(1)如果其中至少有一个是布尔值，那么会隐式的调用Number()进行转换，然后比较。
(2)如果一个为字符串。另一个为数值，那么会隐式的调用Number()对字符串进行转换，如果转换不成功，则返回false;

三、语句类
  （1） if(表达式){ }else{ }
  （2） var  变量=  Boolean expression?真值:假值
    
    if语句和三元表达式里面的表达式会隐式的调用Boolean()函数，按照这个函数的转换规则，转换为相应的布尔值
